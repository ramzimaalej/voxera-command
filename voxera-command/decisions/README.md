# Decision log (ADRs)

One file per real decision. Copy `_template.md`, increment the number, fill it in.

- Numbers are permanent. Don't renumber.
- Don't delete superseded ADRs — set `status: superseded` and link `superseded_by`.
- This is the strategic "why". Babysitter's per-run journal (`.a5c/runs/`) is the execution "what". Different layers; keep both.
